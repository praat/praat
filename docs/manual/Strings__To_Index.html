Strings: To Index

Generates an Index from the selected Strings.

Example

We start from the following Strings:

6 (number of strings)
 "hallo"
 "dag allemaal"
 "hallo"
 "tot morgen"
 "hallo"
 "tot morgen"

This will give us the following Index:

1 (number of columns) "" (no column name)
 "dag allemaal"
 "hallo"
 "tot morgen"
 6 (number of elements)

 1
