FAQ: How to cite Praat

Question: how do I cite Praat in my articles?

Answer: nowadays most journals allow you to cite computer programs and web sites. The style approved by the American Psychological Association, and therefore by many journals, is like the following (change the dates and version number as needed):

=>
Boersma, Paul & Weenink, David (2025). Praat: doing phonetics by computer [Computer program]. Version 6.4.37, retrieved 28 June 2025 from https://www.praat.org/

If the journal does not allow you to cite a web site, then try:

Boersma, Paul (2001). Praat, a system for doing phonetics by computer. Glot International 5:9/10, 341-345.

This paper can be downloaded from Boersma’s website.
