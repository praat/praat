Dynamic menu

A column of buttons in the right-hand part of the Objects window, plus the Save menu in the Objects window.

If you select one or more objects in the list, the possible actions that you can perform with the selected objects will appear in the dynamic menu. These actions can include viewing & editing, saving, drawing, conversions to other types (including analysis and synthesis), and more.

Example of analysis:

Record a Sound, select it, and click To Pitch.... This will create a new Pitch object and put it in the list of objects. You can then edit, write, and draw this Pitch object.

Example of synthesis:

Create a Speaker, create and edit an Artword, and click To Sound....
