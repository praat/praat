Undo

One of the commands in the Edit menu of the Picture window.

It erases your most recently created drawing, which could have come from a command in the Objects window or from one of the drawing commands in the World and Margins menus.

Behaviour

This command will erase some drawings, but it will not change your settings in the Pen and Font menus (line type, line width, font, font size, and colour).

The world window will be what it was after the latest-but-one drawing, so that you can use the Margins menu as if the latest drawing had never happened.
