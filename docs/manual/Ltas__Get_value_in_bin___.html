Ltas: Get value in bin...

A query to the selected Ltas object.

Return value

the LTAS value (in dB) in a specified bin. If the bin number is less than 1 or greater than the number of bin, the result is 0; otherwise, it is z [1] [bin number].

Setting

Bin number
the bin whose value is to be looked up.
