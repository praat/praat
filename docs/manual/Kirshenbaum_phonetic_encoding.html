Kirshenbaum phonetic encoding

The Kirshenbaum phonetic encoding represents International Phonetic Alphabet symbols using ascii characters. See: http://www.kirshenbaum.net/IPA/ascii-ipa.pdf. The eSpeak speech synthesizer on which our synthesizer is based accepts this encoding as text input.
