Matrix: To TableOfReal

A command to convert every selected Matrix to a TableOfReal.

This command is available from the Cast menu. The resulting TableOfReal has the same number of rows and columns as the original Matrix, and the same data in the cells. However, it does not yet have any row or column labels; you can add those with some commands from the TableOfReal Modify menu.
